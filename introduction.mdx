# TMT API

TMT API is a robust Firebase Cloud Functions backend that powers ticket management and event ticketing platforms. Built for scalability and flexibility, it provides comprehensive APIs for managing events, tickets, orders, payments, and access control.

## Use Cases

TMT API is designed for:

- **Event Organizers**: Manage large-scale events with thousands of attendees
- **Venue Operators**: Control access, track capacity, and manage seating
- **Ticket Distributors**: Sell tickets through multiple channels with real-time inventory
- **Mobile Apps**: Provide seamless ticket purchasing and access via mobile devices

## Architecture

The TMT API is built on Firebase Cloud Functions, leveraging:

- **Firebase Admin SDK** for authentication and Firestore database operations
- **PostgreSQL** for transactional data and reporting
- **Cloud Functions v2** for serverless compute and scalability
- **Third-party integrations** for payments, seating, and billing

<Note>
  All functions are exposed as HTTPS callable endpoints, secured with Firebase Authentication.
</Note>
