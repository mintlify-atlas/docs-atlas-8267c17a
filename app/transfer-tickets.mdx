## Overview

The `app_transfer_tickets` endpoint allows users to transfer ticket ownership to another user. This is useful for gifting tickets or reselling through authorized channels.

## Request

### Endpoint

```
POST https://us-central1-{project}.cloudfunctions.net/app_transfer_tickets
```

### Headers

| Header | Value |
|--------|-------|
| Content-Type | `application/json` |
| Authorization | `Bearer {firebase_id_token}` |

### Body Parameters

<ParamField path="from_user_id" type="string" required>
  Current owner's user ID (must match authenticated user)
</ParamField>

<ParamField path="to_user_email" type="string" required>
  Recipient's email address (must be registered in app_users)
</ParamField>

<ParamField path="ticket_ids" type="array" required>
  Array of ticket IDs to transfer
</ParamField>

<ParamField path="message" type="string">
  Optional message to recipient
</ParamField>

```javascript JavaScript
const transferTickets = async (fromUserId, toEmail, ticketIds, message = '') => {
  const response = await fetch(
    'https://us-central1-{project}.cloudfunctions.net/app_transfer_tickets',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${idToken}`
      },
      body: JSON.stringify({
        data: {
          from_user_id: fromUserId,
          to_user_email: toEmail,
          ticket_ids: ticketIds,
          message
        }
      })
    }
  );
  
  return await response.json();
};

const result = await transferTickets(
  'user_abc123',
  'friend@example.com',
  ['evt_xyz789-tk_1', 'evt_xyz789-tk_2'],
  'Enjoy the concert!'
);
```

```swift iOS (Swift)
import FirebaseFunctions

func transferTickets(
    fromUserId: String,
    toEmail: String,
    ticketIds: [String],
    message: String = ""
) async throws -> TransferResult {
    let functions = Functions.functions()
    let data: [String: Any] = [
        "from_user_id": fromUserId,
        "to_user_email": toEmail,
        "ticket_ids": ticketIds,
        "message": message
    ]
    
    let result = try await functions.httpsCallable("app_transfer_tickets").call(data)
    guard let resultData = result.data as? [String: Any] else {
        throw TransferError.invalidResponse
    }
    
    return TransferResult(dictionary: resultData)
}

let result = try await transferTickets(
    fromUserId: user.uid,
    toEmail: "friend@example.com",
    ticketIds: ["evt_xyz789-tk_1"],
    message: "Enjoy!"
)
```

## Transfer Rules

<Warning>
  Tickets cannot be transferred if:
  - Already used for venue entry (`access_status: true`)
  - Event has already started or ended
  - Ticket is locked during checkout
  - Recipient email is not registered
</Warning>

### Validation Checks

The system validates:

1. **Ownership**: Sender must own all tickets
2. **Availability**: Tickets must be transferable
3. **Recipient**: Must have valid app_users account
4. **Event Status**: Event must be active and not started
5. **Access Status**: Tickets must not have been used

## Transfer Flow

<Steps>
  <Step title="Validate Ownership">
    Verify sender owns all tickets in `ticket_ids` array.
  </Step>
  
  <Step title="Check Transferability">
    Ensure tickets meet transfer requirements:
    ```javascript
    const canTransfer = ticket => 
      !ticket.access_status && 
      ticket.transferable &&
      new Date(ticket.event_date) > new Date();
    ```
  </Step>
  
  <Step title="Find Recipient">
    Look up recipient by email in `app_users` collection.
  </Step>
  
  <Step title="Update Tickets">
    Update customer information and add ledger entry:
    ```javascript
    await db.collection('events').doc(eventId)
      .collection('tickets').doc(ticketId)
      .update({
        'customer.id': toUserId,
        'customer.email': toEmail,
        'customer.name': toUserName,
        ledger: FieldValue.arrayUnion({
          action: 'transferred',
          date: Timestamp.now(),
          metadata: {
            from_user: fromUserId,
            to_user: toUserId,
            transfer_id: transferId
          }
        })
      });
    ```
  </Step>
  
  <Step title="Send Notifications">
    Notify both sender and recipient via email/push notification.
  </Step>
</Steps>

## Offline Transfer

For offline scenarios, use `app_transfer_tickets_offline`:

```javascript
const { app_transfer_tickets_offline } = require('./functions/app_users/app_user');

const result = await app_transfer_tickets_offline({
  from_user_id: 'user_abc123',
  to_user_email: 'friend@example.com',
  ticket_ids: ['evt_xyz789-tk_1'],
  offline_timestamp: '2026-03-20T10:30:00Z'
});

// Sync when back online
const syncResult = await syncOfflineTransfers();
```

<Note>
  Offline transfers are queued and processed when connectivity is restored.
</Note>

## Mobile App Implementation

### Transfer UI Component

```javascript
import React, { useState } from 'react';
import { View, Text, TextInput, Button, Alert, StyleSheet } from 'react-native';

const TransferTicketScreen = ({ ticket, userId, onTransferComplete }) => {
  const [recipientEmail, setRecipientEmail] = useState('');
  const [message, setMessage] = useState('');
  const [loading, setLoading] = useState(false);
  
  const handleTransfer = async () => {
    if (!recipientEmail) {
      Alert.alert('Error', 'Please enter recipient email');
      return;
    }
    
    setLoading(true);
    try {
      const result = await transferTickets(
        userId,
        recipientEmail,
        [ticket.ticket_id],
        message
      );
      
      if (result.data.transferred) {
        Alert.alert(
          'Success',
          `Ticket transferred to ${result.data.to_user.name}`,
          [{ text: 'OK', onPress: onTransferComplete }]
        );
      }
    } catch (error) {
      Alert.alert('Error', error.message);
    } finally {
      setLoading(false);
    }
  };
  
  const canTransfer = !ticket.access_status && 
    ticket.transferable &&
    new Date(ticket.event_date) > new Date();
  
  if (!canTransfer) {
    return (
      <View style={styles.container}>
        <Text style={styles.warning}>
          This ticket cannot be transferred
        </Text>
      </View>
    );
  }
  
  return (
    <View style={styles.container}>
      <Text style={styles.title}>Transfer Ticket</Text>
      
      <View style={styles.ticketInfo}>
        <Text style={styles.eventName}>{ticket.event_name}</Text>
        <Text>{ticket.zone} - {ticket.seat_id}</Text>
      </View>
      
      <TextInput
        style={styles.input}
        placeholder="Recipient Email"
        value={recipientEmail}
        onChangeText={setRecipientEmail}
        keyboardType="email-address"
        autoCapitalize="none"
      />
      
      <TextInput
        style={[styles.input, styles.messageInput]}
        placeholder="Message (optional)"
        value={message}
        onChangeText={setMessage}
        multiline
        numberOfLines={4}
      />
      
      <Button
        title={loading ? "Transferring..." : "Transfer Ticket"}
        onPress={handleTransfer}
        disabled={loading}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    marginBottom: 20
  },
  ticketInfo: {
    backgroundColor: '#f5f5f5',
    padding: 16,
    borderRadius: 8,
    marginBottom: 20
  },
  eventName: {
    fontSize: 18,
    fontWeight: '600',
    marginBottom: 4
  },
  input: {
    borderWidth: 1,
    borderColor: '#ddd',
    borderRadius: 8,
    padding: 12,
    marginBottom: 16,
    fontSize: 16
  },
  messageInput: {
    height: 100,
    textAlignVertical: 'top'
  },
  warning: {
    color: '#f44336',
    fontSize: 16,
    textAlign: 'center'
  }
});

export default TransferTicketScreen;
```
