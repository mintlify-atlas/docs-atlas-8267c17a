## Overview

The `app_user_validate` endpoint authenticates mobile app users using their email and a security code. It returns user information and authentication status.

## Request

### Endpoint

```
POST https://us-central1-{project}.cloudfunctions.net/app_user_validate
```

### Headers

| Header | Value |
|--------|-------|
| Content-Type | `application/json` |
| Authorization | `Bearer {firebase_id_token}` |

### Body Parameters

<ParamField path="email" type="string" required>
  User's email address
</ParamField>

<ParamField path="code" type="string" required>
  Security code sent to user (4-6 digit code)
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST \
  https://us-central1-{project}.cloudfunctions.net/app_user_validate \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer {token}' \
  -d '{
    "data": {
      "email": "john@example.com",
      "code": "123456"
    }
  }'
```

```javascript JavaScript
const validateUser = async (email, code) => {
  const response = await fetch(
    'https://us-central1-{project}.cloudfunctions.net/app_user_validate',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${idToken}`
      },
      body: JSON.stringify({
        data: { email, code }
      })
    }
  );
  
  return await response.json();
};

const result = await validateUser('john@example.com', '123456');
console.log(result);
```

```python Python
import requests

def validate_user(email, code, token):
    url = 'https://us-central1-{project}.cloudfunctions.net/app_user_validate'
    headers = {
        'Content-Type': 'application/json',
        'Authorization': f'Bearer {token}'
    }
    data = {
        'data': {
            'email': email,
            'code': code
        }
    }
    
    response = requests.post(url, headers=headers, json=data)
    return response.json()

result = validate_user('john@example.com', '123456', id_token)
print(result)
```

## Validation Flow

<Steps>
  <Step title="Generate Security Code">
    Server generates a 6-digit code and stores it in the user's record:
    
    ```javascript
    const code = Math.floor(100000 + Math.random() * 900000).toString();
    await db.collection('app_users').doc(userId).update({
      security_code: code,
      code_expires_at: Timestamp.fromMillis(Date.now() + 600000) // 10 min
    });
    ```
  </Step>
  
  <Step title="Send Code to User">
    Code is sent via email or SMS using the `send_email` or `send_notifications` endpoint.
  </Step>
  
  <Step title="User Submits Code">
    Mobile app calls `app_user_validate` with email and code.
  </Step>
  
  <Step title="Validation">
    Server checks:
    - Code matches stored value
    - Code hasn't expired
    - User exists in app_users collection
  </Step>
  
  <Step title="Return Result">
    If valid, return user data. If invalid, return error.
  </Step>
</Steps>

## Error Codes

| Code | Description | Resolution |
|------|-------------|------------|
| `USER_NOT_FOUND` | Email not registered | User needs to sign up first |
| `INVALID_CODE` | Security code doesn't match | Request new code |
| `CODE_EXPIRED` | Security code expired | Request new code |
| `ACCOUNT_DISABLED` | User account is disabled | Contact support |
| `INVALID_REQUEST` | Missing email or code | Check request parameters |

## Security Notes

<Warning>
  Security codes should expire after 10 minutes and be single-use only.
</Warning>

<Note>
  After 3 failed validation attempts, consider implementing a lockout period to prevent brute force attacks.
</Note>

## Usage Example

### Mobile App Login Flow

```javascript
// Step 1: Request security code
const requestCode = async (email) => {
  await fetch('https://us-central1-{project}.cloudfunctions.net/helpers_app_pin', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({data: {email}})
  });
};

// Step 2: User enters code from email
const code = await getUserInput(); // From UI input

// Step 3: Validate code
const result = await validateUser(email, code);

if (result.data.valido) {
  // Store user data locally
  await AsyncStorage.setItem('user', JSON.stringify(result.data.user));
  
  // Navigate to home screen
  navigation.navigate('Home');
} else {
  // Show error message
  showError('Invalid code. Please try again.');
}
```
