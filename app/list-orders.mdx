## Overview

The `app_list_orders` endpoint returns all orders associated with a user's account, including order details, tickets, and payment information.

## Request

### Endpoint

```
POST https://us-central1-{project}.cloudfunctions.net/app_list_orders
```

### Headers

| Header | Value |
|--------|-------|
| Content-Type | `application/json` |
| Authorization | `Bearer {firebase_id_token}` |

### Body Parameters

<ParamField path="user_id" type="string" required>
  User ID from Firebase Authentication
</ParamField>

<ParamField path="status" type="string">
  Filter orders by status: `"completed"`, `"processing"`, `"failed"`, or `"all"` (default)
</ParamField>

<ParamField path="limit" type="number">
  Maximum number of orders to return (default: 50)
</ParamField>

```javascript JavaScript
const listOrders = async (userId, status = 'all', limit = 50) => {
  const response = await fetch(
    'https://us-central1-{project}.cloudfunctions.net/app_list_orders',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${idToken}`
      },
      body: JSON.stringify({
        data: { user_id: userId, status, limit }
      })
    }
  );
  
  return await response.json();
};

const orders = await listOrders('user_abc123', 'completed', 20);
console.log(orders);
```

```swift iOS (Swift)
import FirebaseAuth
import FirebaseFunctions

func listOrders(userId: String, status: String = "all") async throws -> [Order] {
    let functions = Functions.functions()
    let data = [
        "user_id": userId,
        "status": status,
        "limit": 50
    ]
    
    let result = try await functions.httpsCallable("app_list_orders").call(data)
    guard let orders = result.data as? [[String: Any]] else {
        throw OrderError.invalidResponse
    }
    
    return orders.compactMap { Order(dictionary: $0) }
}

let orders = try await listOrders(userId: user.uid, status: "completed")
```

## Filtering and Sorting

### Filter by Event

To get orders for specific events:

```javascript
const eventOrders = await listOrders(userId, 'completed')
  .then(orders => orders.data.filter(o => o.event_id === 'evt_xyz789'));
```

### Sort by Date

```javascript
const sortedOrders = orders.data.sort((a, b) => 
  new Date(b.created_at) - new Date(a.created_at)
);
```

## Mobile App Implementation

### React Native Example

```javascript
import React, { useEffect, useState } from 'react';
import { FlatList, View, Text, StyleSheet } from 'react-native';

const OrdersScreen = ({ userId }) => {
  const [orders, setOrders] = useState([]);
  const [loading, setLoading] = useState(true);
  
  useEffect(() => {
    loadOrders();
  }, []);
  
  const loadOrders = async () => {
    try {
      const result = await listOrders(userId, 'completed');
      setOrders(result.data);
    } catch (error) {
      console.error('Failed to load orders:', error);
    } finally {
      setLoading(false);
    }
  };
  
  const renderOrder = ({ item }) => (
    <View style={styles.orderCard}>
      <Text style={styles.eventName}>{item.event_name}</Text>
      <Text>{item.tickets.length} ticket(s)</Text>
      <Text>${item.total.toFixed(2)} {item.currency}</Text>
      <Text style={styles.date}>
        {new Date(item.created_at).toLocaleDateString()}
      </Text>
    </View>
  );
  
  if (loading) return <Text>Loading orders...</Text>;
  
  return (
    <FlatList
      data={orders}
      renderItem={renderOrder}
      keyExtractor={item => item.order_id}
      ListEmptyComponent={<Text>No orders found</Text>}
    />
  );
};

const styles = StyleSheet.create({
  orderCard: {
    padding: 16,
    marginVertical: 8,
    backgroundColor: '#fff',
    borderRadius: 8,
    shadowColor: '#000',
    shadowOpacity: 0.1,
    shadowRadius: 4,
    elevation: 2
  },
  eventName: {
    fontSize: 18,
    fontWeight: 'bold',
    marginBottom: 4
  },
  date: {
    color: '#666',
    marginTop: 4
  }
});

export default OrdersScreen;
```
