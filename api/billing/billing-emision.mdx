## Overview

This endpoint emits electronic invoices to an external billing API. It handles authentication, document generation, and submission to the external billing system (The Factory HKA).

## Authentication

The endpoint internally authenticates with the external billing API using credentials configured in the system.

## Tax Calculation

### VAT (IVA)
- Rate: 16%
- Applied to subtotal
- Calculated with rounding to 2 decimals

### IGTF (Currency Tax)
- Rate: 3%
- Applied only when payment is made in foreign currency (Zelle, Cash Dollars, Foreign Cash)
- Base: Amount paid in foreign currency (capped at invoice total)
- Calculated with truncation to 2 decimals

### Exchange Rate
- USD amounts converted to BSD (Bolívares)
- Individual components truncated to 2 decimals
- Total must equal sum of truncated components

## Error Handling

- **500**: Authentication failed with external API
- **500**: Validation errors in document structure
- **500**: External API error

Error responses include:
```json
{
  "mensaje": "Error description",
  "validaciones": [
    {
      "campo": "field_name",
      "error": "error_description"
    }
  ]
}
```
