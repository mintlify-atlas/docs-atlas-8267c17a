## Error Codes

| Status Code | Description |
| --- | --- |
| 200 | Collaborator successfully created |
| 400 | Error creating user (email already exists, invalid data, etc.) |

<Note>
  - The collaborator is created in both Firebase Authentication and the `u_collaborators` Firestore collection
  - The `client` field must reference a valid client UID
  - Email addresses must be unique across the system
  - The `date.create` field is automatically set to the current timestamp
  - Collaborators are associated with a specific client account
</Note>
