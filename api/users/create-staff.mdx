  </ParamField>

## Error Codes

| Status Code | Description |
| --- | --- |
| 200 | Staff member successfully created |
| 400 | Error creating user (email already exists, invalid data, etc.) |

<Note>
  - The staff member is created in both Firebase Authentication and the `u_staff` Firestore collection
  - Two-factor authentication is disabled by default (can be enabled later)
  - Email addresses must be unique across the system
  - The `date.create` field is automatically set to the current timestamp
  - Staff accounts have administrative privileges and access to platform management features
  - The `security_2fa` object is initialized with `status: false` and `type: "email"`
</Note>
