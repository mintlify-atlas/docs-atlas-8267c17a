## Error Codes

| Status Code | Description |
| --- | --- |
| 200 | Client successfully created |
| 400 | Error creating user (email already exists, invalid data, etc.) |

<Note>
  - The client is created in both Firebase Authentication and the `u_clients` Firestore collection
  - The password must meet Firebase Authentication requirements
  - Email addresses must be unique across the system
  - The `date.create` field is automatically set to the current timestamp
  - Initial counters for collaborators, contracts, and events are set to 0
</Note>
