TMT API supports multiple payment processors to accommodate different markets and customer preferences. This guide covers integrating Stripe (international), Mercantil (Venezuelan banking), and Banco de Venezuela payment methods.

## Supported Payment Methods

The TMT platform supports:

- **Stripe**: Credit/debit cards, international payments
- **Mercantil**: TDD (debit cards), TDC (credit cards), C2P (mobile payments)
- **Banco de Venezuela**: Mobile payments (Pago Móvil)

## Payment Flow

<Steps>
  <Step title="Lock Tickets">
    Reserve tickets during checkout to prevent double-booking.
  </Step>
  
  <Step title="Create Payment Intent">
    Initialize payment with chosen processor.
  </Step>
  
  <Step title="Process Payment">
    Complete payment through gateway.
  </Step>
  
  <Step title="Create Order">
    On successful payment, create order and assign tickets.
  </Step>
  
  <Step title="Release Locks">
    Unlock tickets (automatic on order creation).
  </Step>
</Steps>

## Stripe Integration

Stripe is the recommended payment processor for international customers and credit card payments.

### Create Payment Intent

Initialize a Stripe payment:

```javascript
const createPaymentIntent = {
  data: {
    amount: 150.00,  // Amount in USD
    currency: "usd",
    description: "Summer Music Festival 2024 - 2 VIP Tickets",
    metadata: {
      event_id: "evt_2024_concert_001",
      customer_email: "customer@example.com",
      ticket_count: 2
    }
  }
};

const response = await axios.post(
  'https://your-api-url/stripe_create_payment_intent',
  createPaymentIntent
);
```

### Response

```json
{
  "message": "Procesado",
  "status": 200,
  "data": {
    "success": true,
    "client_secret": "pi_3AbCdEfGhIjKlMnO_secret_1234567890",
    "payment_intent_id": "pi_3AbCdEfGhIjKlMnO"
  }
}
```

<Tip>
Store the `payment_intent_id` in your session. You'll need it to confirm the payment and retrieve payment details.
</Tip>

### Complete Payment (Client-Side)

Use Stripe.js on the frontend to complete the payment:

<CodeGroup>
```javascript React
import { loadStripe } from '@stripe/stripe-js';
import { Elements, PaymentElement, useStripe, useElements } from '@stripe/react-stripe-js';

const stripePromise = loadStripe('pk_live_your_publishable_key');

function CheckoutForm({ clientSecret }) {
  const stripe = useStripe();
  const elements = useElements();

  const handleSubmit = async (event) => {
    event.preventDefault();

    if (!stripe || !elements) return;

    const result = await stripe.confirmPayment({
      elements,
      confirmParams: {
        return_url: 'https://yoursite.com/payment-complete',
      },
    });

    if (result.error) {
      console.error(result.error.message);
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <PaymentElement />
      <button disabled={!stripe}>Pay Now</button>
    </form>
  );
}

function App({ clientSecret }) {
  return (
    <Elements stripe={stripePromise} options={{ clientSecret }}>
      <CheckoutForm clientSecret={clientSecret} />
    </Elements>
  );
}
```

```javascript Vanilla JavaScript
const stripe = Stripe('pk_live_your_publishable_key');

const elements = stripe.elements({
  clientSecret: 'pi_3AbCdEfGhIjKlMnO_secret_1234567890'
});

const paymentElement = elements.create('payment');
paymentElement.mount('#payment-element');

document.getElementById('payment-form').addEventListener('submit', async (e) => {
  e.preventDefault();
  
  const { error } = await stripe.confirmPayment({
    elements,
    confirmParams: {
      return_url: 'https://yoursite.com/payment-complete',
    },
  });
  
  if (error) {
    alert(error.message);
  }
});
```
</CodeGroup>

Get details about a payment intent:

```javascript

  'https://your-api-url/stripe_get_payment_intent',

```

<Warning>
Always verify payment status on your server before creating orders. Never trust client-side confirmation alone.
</Warning>

## Mercantil Integration

Mercantil is a Venezuelan bank that supports multiple payment methods including debit cards (TDD), credit cards (TDC), and mobile payments (C2P).

### TDD (Debit Card) Payment

Process debit card payments:

#### Request Authorization

First, request authorization from the user:

```javascript
const authRequest = {
  data: {
    customer_id: "V12345678",
    card_number: "501878200066287386",
    ipaddress: "192.168.1.100"
  }
};

const response = await axios.post(
  'https://your-api-url/api_mercantil_auth',
  authRequest
);
```

#### Process Payment

After receiving authorization code, process the payment:

```javascript
const tddPayment = {
  data: {
    amount: 1500.00,  // Amount in VES
    invoice_number: "INV-2024-001",
    customer_id: "V12345678",
    card_number: "501878200066287386",
    cvv: "752",
    account_type: "cc",  // cc = current account, ca = savings
    expiration_date: "2026/11",
    twofactor_auth: "11111111",  // Authorization code from user
    ipaddress: "192.168.1.100"
  }
};

const response = await axios.post(
  'https://your-api-url/api_mercantil_tdd_pay',
  tddPayment
);
```

### Response

```json
{
  "message": "Procesado",
  "data": {
    "transaction_id": "TXN123456789",
    "status": "approved",
    "amount": 1500.00,
    "currency": "VES",
    "authorization_code": "AUTH789456",
    "timestamp": "2024-03-11T14:30:00Z"
  }
}
```

### C2P (Mobile Payment)

Process mobile payments (Pago Móvil):

<Steps>
  <Step title="Request Authorization">
    Request authorization code from the mobile payment system.
  </Step>
  
  <Step title="User Approves">
    User approves transaction in their banking app.
  </Step>
  
  <Step title="Submit Payment">
    Complete payment with authorization code.
  </Step>
</Steps>

#### Request Authorization

```javascript
const c2pAuth = {
  data: {
    destination_id: "V18367443",
    destination_mobile_number: "584241513063",
    ipaddress: "192.168.1.100"
  }
};

const response = await axios.post(
  'https://your-api-url/api_mercantil_c2p_auth',
  c2pAuth
);
```

#### Process Payment

```javascript
const c2pPayment = {
  data: {
    amount: 1800.00,
    invoice_number: "INV-2024-003",
    destination_bank_id: "0105",  // Mercantil bank code
    destination_id: "V18367443",
    destination_mobile_number: "584241513063",
    origin_mobile_number: "584126935529",
    payment_reference: "REF123",
    twofactor_auth: "00001111",
    ipaddress: "192.168.1.100"
  }
};

const response = await axios.post(
  'https://your-api-url/api_mercantil_c2p_pay',
  c2pPayment
);
```

### Search C2P Transaction

Search for a mobile payment transaction:

```javascript
const searchPayment = {
  data: {
    destination_mobile_number: "584241513063",
    origin_mobile_number: "584126935529",
    payment_reference: "REF123",
    date_pay: "2024-03-11",
    amount: 1800.00,
    ipaddress: "192.168.1.100"
  }
};

const response = await axios.post(
  'https://your-api-url/api_mercantil_c2p_search',
  searchPayment

```

<Tip>
All sensitive data (CVV, authorization codes, phone numbers) are automatically encrypted before transmission using AES-256 encryption.
</Tip>

## Complete Payment Flow Example

```javascript
const processTicketPurchase = async (tickets, customerInfo) => {
  try {
    // 1. Lock tickets
    const lockResponse = await axios.post(
      'https://your-api-url/tickets_lock',
      {
        data: {
          ticketd_reserved: tickets.map(t => t.ticket_id),
          reserved_time: 900
        }
      }
    );
    
    if (lockResponse.data.status !== 200) {
      throw new Error('Tickets not available');
    }
    
    // 2. Calculate total amount
    const totalAmount = tickets.reduce((sum, t) => sum + t.price, 0);
    
    // 3. Create payment intent
    const paymentIntent = await axios.post(
      'https://your-api-url/stripe_create_payment_intent',
      {
        data: {
          amount: totalAmount,
          currency: 'usd',
          metadata: {
            event_id: tickets[0].event_id,
            customer_email: customerInfo.email
          }
        }
      }
    );
    
    // 4. Process payment (client-side with Stripe.js)
    // ... Stripe payment flow ...
    
    // 5. Confirm payment
    const confirmation = await axios.post(
      'https://your-api-url/stripe_confirm_payment',
      {
        data: {
          payment_intent_id: paymentIntent.data.data.payment_intent_id
        }
      }
    );
    
    if (confirmation.data.data.payment_intent.status !== 'succeeded') {
      throw new Error('Payment failed');
    }
    
    // 6. Create order
    const order = await axios.post(
      'https://your-api-url/order_created',
      {
        data: {
          amount: totalAmount,
          tickets: tickets,
          transactions: [{
            amount: totalAmount,
            payment_id: 'stripe',
            payment_data: paymentIntent.data.data.payment_intent_id,
            status: true
          }],
          // ... other order data
        }
      }
    );
    
    return {
      success: true,
      order_id: order.data.data.order
    };
    
  } catch (error) {
    // Unlock tickets on error
    await axios.post(
      'https://your-api-url/tickets_unlock_param',
      {
        data: {
          ticketd_reserved: tickets.map(t => t.ticket_id)
        }
      }
    );
    
    return {
      success: false,
      error: error.message
    };
  }
};
```
