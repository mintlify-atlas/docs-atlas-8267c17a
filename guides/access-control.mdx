Access control is essential for managing event entry, exit, and overall security. This guide covers ticket validation, entry/exit tracking, credential management, and checkpoint configuration.

## Access Control Components

The TMT access control system includes:

- **Ticket Scanning**: Validate tickets at entry points
- **Entry/Exit Tracking**: Monitor attendee movement
- **Credentials**: Manage staff and VIP access
- **Checkpoints**: Configure multiple access points
- **Offline Sync**: Support for offline operations

## Access Control Flow

<Steps>
  <Step title="Scan Ticket">
    Scan the ticket QR code or enter ticket ID manually.
  </Step>
  
  <Step title="Validate Ticket">
    System checks ticket validity and purchase status.
  </Step>
  
  <Step title="Record Entry">
    Log entry with timestamp and checkpoint information.
  </Step>
  
  <Step title="Update Ticket Status">
    Mark ticket as accessed in ledger.
  </Step>
  
  <Step title="Allow/Deny Entry">
    Grant or deny access based on validation result.
  </Step>
</Steps>

## Ticket Entry Control

### Check In (Entry)

Validate and record ticket entry:

```javascript
const ticketEntry = {
  data: {
    ticket_id: "evt_2024_concert_001-abc123xyz"
  }
};

const response = await axios.post(
  'https://your-api-url/tickets_access_control_in',
  ticketEntry
);
```

### Entry Validation Logic

The system performs these checks:

1. **Ticket Exists**: Verify ticket ID is valid
2. **Ticket Sold**: Confirm ticket has been purchased (`status: false`)
3. **Not Previously Used**: Check if ticket already accessed
4. **Re-entry Allowed**: Allow re-entry if user exited previously

## Access Status Tracking

Each ticket has two access flags:

- `access_status`: Has the ticket been used at least once?
- `access_entry`: Is the ticket holder currently inside the venue?

### Status States

| access_status | access_entry | State |
|---------------|--------------|-------|
| false | false | Never entered |
| true | true | Currently inside |
| true | false | Inside then left (can re-enter) |

## Building an Access Control Interface

Here's a complete example of an access control scanner:

```javascript
class AccessControlScanner {
  constructor(eventId) {
    this.eventId = eventId;
    this.offlineQueue = [];
  }
  
  async scanTicket(ticketId) {
    try {
      const response = await axios.post(
        'https://your-api-url/tickets_access_control_in',
        {
          data: { ticket_id: ticketId }
        }
      );
      
      if (response.data.status === 200) {
        this.displaySuccess(response.data.data.message);
        this.logEntry(ticketId, 'in');
        return true;
      } else {
        this.displayError(response.data.data.message);
        return false;
      }
    } catch (error) {
      // Network error - add to offline queue
      this.offlineQueue.push({
        ticket_id: ticketId,
        date: new Date().toISOString(),
        tipo: 'in'
      });
      this.displayWarning('Offline mode - scan queued');
      return true; // Allow entry in offline mode
    }
  }
  
  async scanExit(ticketId) {
    try {
      const response = await axios.post(
        'https://your-api-url/tickets_access_control_out',
        {
          data: { ticket_id: ticketId }
        }
      );
      
      if (response.data.status === 200) {
        this.displaySuccess('Exit recorded');
        this.logEntry(ticketId, 'out');
        return true;
      }
    } catch (error) {
      this.offlineQueue.push({
        ticket_id: ticketId,
        date: new Date().toISOString(),
        tipo: 'out'
      });
      return true;
    }
  }
  
  async syncOfflineScans() {
    if (this.offlineQueue.length === 0) return;
    
    try {
      const response = await axios.post(
        'https://your-api-url/tickets_access_control_cold',
        {
          data: {
            tickets_cold: this.offlineQueue
          }
        }
      );
      
      if (response.data.status === 200) {
        console.log(`Synced ${this.offlineQueue.length} offline scans`);
        this.offlineQueue = [];
      }
    } catch (error) {
      console.error('Sync failed, will retry later');
    }
  }
  
  displaySuccess(message) {
    console.log('✓ ' + message);
    // Update UI with green indicator
  }
  
  displayError(message) {
    console.error('✗ ' + message);
    // Update UI with red indicator
  }
  
  displayWarning(message) {
    console.warn('⚠ ' + message);
    // Update UI with yellow indicator
  }
  
  logEntry(ticketId, direction) {
    const log = {
      ticket_id: ticketId,
      direction: direction,
      timestamp: new Date().toISOString()
    };
    // Store in local database for reporting
  }
}

// Usage
const scanner = new AccessControlScanner('evt_2024_concert_001');

// Scan ticket
await scanner.scanTicket('evt_2024_concert_001-abc123xyz');

// Periodically sync offline scans
setInterval(() => {
  scanner.syncOfflineScans();
}, 60000); // Every minute
```

## Access Control Analytics

### Real-Time Attendance

```javascript
const getAttendance = async (eventId) => {
  const tickets = await db.collection('events')
    .doc(eventId)
    .collection('tickets')
    .where('status', '==', false)
    .get();
  
  let totalSold = 0;
  let currentlyInside = 0;
  let totalEntered = 0;
  
  tickets.forEach(ticket => {
    const data = ticket.data();
    totalSold++;
    if (data.access_status) totalEntered++;
    if (data.access_entry) currentlyInside++;
  });
  
  return {
    total_sold: totalSold,
    total_entered: totalEntered,
    currently_inside: currentlyInside,
    entry_rate: (totalEntered / totalSold * 100).toFixed(1) + '%'
  };
};
```

### Entry Rate by Zone

```javascript
const getEntryRateByZone = async (eventId) => {
  const tickets = await db.collection('events')
    .doc(eventId)
    .collection('tickets')
    .where('status', '==', false)
    .get();
  
  const zones = {};
  
  tickets.forEach(ticket => {
    const data = ticket.data();
    if (!zones[data.zone]) {
      zones[data.zone] = { sold: 0, entered: 0 };
    }
    zones[data.zone].sold++;
    if (data.access_status) zones[data.zone].entered++;
  });
  
  return Object.keys(zones).map(zone => ({
    zone: zone,
    sold: zones[zone].sold,
    entered: zones[zone].entered,
    entry_rate: (zones[zone].entered / zones[zone].sold * 100).toFixed(1) + '%'
  }));
};
```

```javascript
// Allow re-entry
const allowReentry = async (ticketId) => {
  // User exits
  await scanner.scanExit(ticketId);
  
  // Later, user returns
  await scanner.scanTicket(ticketId);
  // System allows re-entry automatically
};
```

```javascript
const preventDuplicate = async (ticketId) => {
  const response = await axios.post(
    'https://your-api-url/tickets_access_control_in',
    { data: { ticket_id: ticketId } }
  );
  
  if (response.data.status === 400) {
    // Ticket already inside

    // Sound alarm or notify security
    securityAlert(ticketId);
  }
};
```
